
# Movie Lib

O Movie Lib é uma biblioteca de filmes consumindo do TMDB API.

````
Acesse em: 

https://movieslib-plum.vercel.app/
````

## Funcionalidades

- Leitura de filmes
- Leitura de filmes por ID
- Busca por filmes


## Stack utilizada

**Front-end:** React, Vite
